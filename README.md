# PSTAT 174 (Time Series)

## Temporal Patterns in Populatr Spotify Songs (2015-2025)

A time series analysis project that aims to identify whether teh frequency of song releases exhibits trend, season, or cyclical patterns over time. A monthly time series is constructed by aggregating song counts and the Box-Jenkind approach is applied to estimate a SARIMA model. Spectral analysis is also applied to examine the frequency structure.
